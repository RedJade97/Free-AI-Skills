# Free AI Skills

A growing collection of free **Claude Skills** — zero-install AI tools for small business owners, operators, and trades. Each skill is a single `.md` file you paste into your own Claude account. No sign-up, no server, no API key. Your data stays in your chat.

Grab what you need, ignore the rest.

## How the "paste into Claude" workflow works

1. You open the tool's `SKILL.md` file on GitHub.
2. You copy the contents.
3. You paste them into a Claude project's Custom Instructions (or into any new chat as the first message).
4. Claude now behaves as that tool — it walks you through a questionnaire, produces the output, and can export to PDF or draft an email.

Each tool's README has the exact steps.
